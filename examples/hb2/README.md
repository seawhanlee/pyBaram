# HB-2 model example

This directory contains example input files for the HB-2 axisymmetric body at
Mach 2.0.

The three-dimensional example uses `hb2.ini`. The mesh file can be downloaded
from:

https://drive.google.com/file/d/1S5w4QfxIzi2Z74Tnh9wZPBT-OQSclq8D

The `axi` directory contains the two-dimensional axisymmetric example:

- `axi/hb2axi.cgns`
- `axi/hb2axi.ini`

See the pyBaram examples documentation for run commands and case details.
